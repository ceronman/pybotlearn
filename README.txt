DEPENDENCIES:

You might need to install some of these before running the game:

  Python:     http://www.python.org/
  PyGTK:     http://www.pygtk.org/


RUNNING THE GAME:

Open a terminal / console and "cd" to the game directory and run:

  python main.py



HOW TO PLAY THE GAME:

PyBotLearn is not exactly a game. Is more an educational tool to teach programing to children.
Currently PyBotLearn is incomplete. You can experiment with the console in the bottom. Try this commands:

move_left()
move_right()
move_up()
move_down()
say('some text here')
variable = ask('some question here ?')

You can also experiment with the arrow buttons at the left.
